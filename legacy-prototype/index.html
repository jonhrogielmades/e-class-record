Capstone Prototype

Track class lists, attendance, and grades in one glass-style workspace.

This e-class-record system converts the project into a role-based academic record platform with two user roles: teachers managing sections, student profiles, attendance, and grading, and students reviewing their own class details, attendance history, and grade summary.

Teacher Role

Manage sections, monitor attendance, enter quizzes and exam grades, and review per-section performance from one dashboard.

Student Role

View assigned section details, attendance records, recent assessments, and a personal grade summary with the same visual system.

Class List

Section A and Section B views show roster counts, adviser information, schedules, and student performance snapshots.

Student Records

Teachers can review learner profiles and attendance entries, while students get a personal records page filtered to their own data.

Grading Module

Input quiz or exam scores, review grade summary tables, and show student-friendly assessment history without using a backend.

Demo Accounts
